## 🎯 ميزات الواجهة الأمامية

### اختيار الطور
- 🏫 **ابتدائي**: السنوات 1-5 + 8 مواد
- 📚 **متوسط**: السنوات 1-4 + 8 مواد
- 🎓 **ثانوي**: جذع مشترك + سنة 2 + سنة 3 (علوم/آداب)

### المواد حسب الطور
| الطور | المواد المتاحة |
|-------|---------------|
| ابتدائي | رياضيات، عربية، فرنسية، إنجليزية، علوم، إسلامية، مدنية، تاريخ/جغرافيا |
| متوسط | رياضيات، علوم، عربية، فيزياء/كيمياء، تاريخ/جغرافيا، فرنسية، إنجليزية، إسلامية |
| ثانوي علوم | رياضيات، فيزياء/كيمياء، علوم، عربية، تاريخ/جغرافيا، فرنسية، إنجليزية، إسلامية |
| ثانوي آداب | عربية، تاريخ/جغرافيا، فرنسية، إنجليزية، إسلامية، فلسفة |

## 📝 معايير الجودة

### تنويع أنواع الأسئلة (Bloom's Taxonomy)
- **MCQ (43%)**: المعرفة، التذكر
- **تطبيقي (36%)**: التطبيق، التحليل
- **مقالي (18%)**: الفهم، الشرح
- **صح/خطأ (2%)**: التحقق السريع

### السياق الجزائري
- 80+ سياقاً جزائرياً (مدن، تاريخ، اقتصاد، بيئة)
- أمثلة من الحياة اليومية
- ربط بالواقع المحلي

### التحقق الآلي
- `validate_question()`: التحقق من الحقول
- `classify_question_difficulty()`: تصنيف الصعوبة
- `response_format="json_object"`: فرض JSON
