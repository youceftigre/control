# خطة التحديث الدوري - الجدول الزمني

## التحديثات الأسبوعية
| اليوم | المهمة | المسؤول |
|-------|--------|---------|
| السبت | جمع ملاحظات المعلمين | مشرف المادة |
| الأحد | تحليل نسب النجاح | فريق الإحصاء |
| الاثنين | تحديد الأسئلة التي تحتاج تعديل | لجنة الجودة |
| الثلاثاء | تنفيذ التعديلات | كاتب الأسئلة |
| الأربعاء | مراجعة التعديلات | أستاذ المادة |
| الخميس | اختبار تجريبي | 20-30 تلميذ |
| الجمعة | اعتماد النهائي | لجنة الاختبارات |

## التحديثات الشهرية
- إضافة 5-10 أسئلة جديدة لكل مادة
- تحديث الأسئلة المرتبطة بالأحداث الجارية
- مراجعة الأسئلة التي حصلت على تقييم <25

## التحديثات الفصلية
- تغطية الوحدات الجديدة المضافة للمنهاج
- إعادة توازن الصعوبة حسب نتائج الفصل
- تحديث السياقات (أسعار، إحصائيات، أحداث)

## التحديثات السنوية
- مراجعة شاملة لجميع الأسئلة (100%)
- تحديث بناءً على تغييرات المنهاج الرسمي
- إضافة مواد جديدة إن لزم الأمر
- أرشفة الأسئلة القديمة
