## 🚀 طريقة النشر على Render

### الخطوة 1: إعداد GitHub
```bash
# 1. انسخ المشروع على GitHub (أو فورك)
# 2. أضف الملفات الآتية:
#    - app.py (خادم Flask)
#    - requirements.txt
#    - templates/index.html
#    - render.yaml
#    - data/questions_full_bank.json
#    - data/subjects_config.json
```

### الخطوة 2: نشر على Render
```bash
1. اذهب إلى https://render.com
2. سجل الدخول بحسابك (أو أنشئ حساب جديد)
3. اضغط على "New +" في الأعلى
4. اختر "Web Service"
5. اربط مستودع GitHub الخاص بك
6. Render سيقرأ تلقائياً render.yaml ويضبط الإعدادات
```

### الخطوة 3: إضافة متغيرات البيئة
```bash
في لوحة التحكم Render:
1. اذهب إلى "Environment"
2. أضف متغير جديد:
   - Key: GROQ_API_KEY
   - Value: [مفتاح Groq الخاص بك]
3. اضغط "Save"
```

### الخطوة 4: الحصول على Groq API Key
```bash
1. اذهب إلى https://console.groq.com
2. سجل الدخول أو أنشئ حساب
3. اضغط على "API Keys" في الشريط الجانبي
4. انسخ المفتاح الخاص بك
5. ألصقه في بيئة Render
```

## 🔧 التثبيت المحلي (للتطوير)

### المتطلبات
- Python 3.8+
- pip (مدير الحزم)
- Git

### خطوات التثبيت
```bash
# 1. انسخ المستودع
git clone https://github.com/yourusername/exam-platform-algeria.git
cd exam-platform-algeria

# 2. أنشئ بيئة افتراضية
python -m venv venv

# 3. فعّل البيئة الافتراضية
# على Linux/Mac:
source venv/bin/activate
# على Windows:
venv\Scripts\activate

# 4. ثبّت المتطلبات
pip install -r requirements.txt

# 5. أضف مفتاح Groq API
export GROQ_API_KEY="your-groq-api-key"  # Linux/Mac
set GROQ_API_KEY=your-groq-api-key       # Windows

# 6. شغّل التطبيق
python app.py
# الآن يمكنك الوصول إلى http://localhost:5000
```
